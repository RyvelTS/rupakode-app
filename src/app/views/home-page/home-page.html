Fitur Utama {{appName}}

Generator Palet Warna Angular Material

Hasilkan palet warna yang konsisten dan siap pakai untuk desain UI/UX Anda, mengikuti standar palet Angular Material.

  • ✔

    Input warna dasar (hex color) untuk berbagai palet (primary, accent, warn, dll).

  • ✔

    Hasilkan rentang warna dari 0 hingga 100, termasuk variasi seperti 4, 6, 12, dll.

  • ✔

    Salin kode warna dalam format yang sesuai untuk Sass/CSS.

  • ✔

    Lihat preview visual dari setiap gradasi warna dalam palet.

Generator Pesan Commit Git Konvensional

Buat pesan commit Git yang terstruktur dan informatif sesuai spesifikasi Conventional Commits 1.0.0.

  • ✔

    Pilih type commit (misal: feat untuk fitur baru, fix untuk perbaikan bug).

  • ✔

    Tambahkan scope opsional untuk konteks (feat(auth)).

  • ✔

    Tuliskan description singkat dan body opsional yang lebih detail.

  • ✔

    Sertakan footer seperti BREAKING CHANGE untuk perubahan mayor.
